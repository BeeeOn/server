Detailed Description

Group represents a number consisting of selected bits of the bitmap type. Picking the group bits from the given raw value creates a number consisting of only group bits.

Example:

  • group: {0, 2, 1}
  • input: 0b1010
  • output: 0b100 (mapping: 0->0 [0], 1->2 [1], 2->1 [0])
