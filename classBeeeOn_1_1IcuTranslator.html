Detailed Description

Utilize icu::ResourceBundle and icu::MessageFormat for localization of messages. Each message is identified by a key and accepts a number of arguments. The arguments are formatted into the message matching the key and translator's locale. The formatting rules are described at the following site:

http://icu-project.org/apiref/icu4c/classicu_1_1MessageFormat.html
