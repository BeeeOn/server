Detailed Description

PocoGlobDBInitializer initializes database with a sequence of individual scripts. The scripts are sorted by names, thus naming convention like 0001-new-schema.sql, 0002-new-table.sql is recommended.

Each script is first preprocessed by the set of preconfigured preprocessors.
