Detailed Description

GatewayMessage represents a message related somehow to a gateway. The message informs its users about some issues or events occuring at the gateway or when communicating with the gateway.

The message itself is a compound of:

  • severity - error, warn, info
  • key - key to lookup the displayable parametrizable text
  • context - JSON object with parameters of the displayable text

The messages are thus defined by the server as a fixed list in some resource file.
