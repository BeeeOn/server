Detailed Description

UnknownEvaluator implements evaluation of formulas as specified in devices.xml to convert from a raw value given from a sensor into the right semantics (target type).

For the given module, convert the raw value into the value as specified by ModuleInfo. The property ModuleInfo::fromUnknown() is required to be non-empty.
