Sanitize the Base64 formatted input. Convert the contents to UTF-8 and allow only Base64 symbols. Whitespace is denied and thus it must not be included. Padding '=' symbols can be included at the end. The method does NOT perform decoding.

If the separators field is a non-empty string, the input can also contain those characters. This allows to support JSON Web Tokens that consist of a sequence of base64-encoded blocks delimited by dots.

string Sanitize::encoding ( const std::string &  bytes,

const std::string &  inputEncoding = "UTF-8" 
)
static

Sanitize encoding on the given input bytes. The bytes are converted to UTF-8 and checked for invalid characters. The default input encoding is UTF-8, however, it can be changed. The output encoding is always UTF-8.

The sizeLimit is intended to avoid too long inputs. Inputs of size longer then the sizeLimit bytes are never processed. Note that the input can contain less characters then bytes (UTF-8). If the sizeLimit is given as SIZE_LIMIT_NONE then no size check is performed.
