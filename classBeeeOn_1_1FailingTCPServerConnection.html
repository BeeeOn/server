Detailed Description

FailingTCPServerConnection is intended for connections that cannot be handled anyway because the client has closed her connection. It should avoid issues with Poco::Net::TCPServer that seems to be stuck when the particular TCPServerConnectionFactory returns nullptr instead of a working connection. The FailingTCPServerConnection throws the Poco::Net::ConnectionAbortedException.
