string IcuTimeZoneImpl::worldName ( const std::string &  id,
const Locale &  locale,
const Poco::Timestamp &  at 
) const
protected

If the time zone with the given id is from region "001" (world), try to derive its display name. This allows to give names like Central Standard Time, Greenwich Mean Time, etc. The method returns an empty string from non-world time zones. For some time zones, the result is the LONG_GMT format like GMT+1.
