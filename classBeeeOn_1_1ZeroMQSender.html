Detailed Description

ZeroMQSender represents a source of messages. It can only send message but it is unable to receive any. Thus, it supports only the following ZeroMQ socket types:

  • PUB - binds to the endpoint (unreliable in principle),
  • PUSH - connects to the endpoint,
  • PAIR - connects to the endpoint,
  • DEALER - binds to the endpoint.
