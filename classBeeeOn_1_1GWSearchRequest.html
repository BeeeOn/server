Detailed Description

This message is used to request searching for a device by criteria. This is a kind of a discovery process where it is known that a certain device has an exact identification. The search is always intended for 1 IoT technology.

Example message: 

{
  "message_type": "search_request",
  "id": "b3f83259-8acf-486d-9ab6-cf74b74ae311",
  "duration": 30,
  "device_prefix": "jablotron",
  "criteria": {
    "serial_number": "1497923740923"
  }
}
