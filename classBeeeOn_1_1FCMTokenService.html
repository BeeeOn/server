Detailed Description

FCMTokenService class provides interface for unregistering and registering users tokens. Every user that wants to receive notifications from BeeeOn server is registered by an app with Firebase Cloud Messaging (FCM) notification service. The FCM provides an authorization token to each user. The received token is then registered with the BeeeOn server. To cancel receiving of notifications from the BeeeOn server, the token can be unregistered.

See also
https://firebase.google.com/docs/cloud-messaging/
