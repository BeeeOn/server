void IcuTimeZoneProvider::listTimeZones ( struct std::vector< TimeZone > &  zones)
overridevirtual

List all time zones via icu::TimeZone::createTimeZoneIDEnumeration() with UCAL_ZONE_TYPE_CANONICAL_LOCATION parameter. This should provide only timezones related to a location like Europe/Prague, Pacific/Tahiti, etc.

In case of a failure from createTimeZoneIDEnumeration() it fallbacks to calling icu::TimeZone::createEnumeration().
